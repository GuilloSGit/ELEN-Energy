# ELEN Energy

## Descripción

ELEN Energy es una plataforma que permite a los usuarios gestionar compras de productos Europeos en Argentina.

## Objetivo

El objetivo de ELEN Energy es facilitar la compra de productos europeos en Argentina, permitiendo a los usuarios acceder a una amplia gama de productos europeos con un solo clic.

## Características

- Tiene aún alto grado de implicancia de atención manual.
- Procesos automatizados para conseguir la información de los productos mediante scraping y saneamiento.
- Control coherente de flujos de datos.
- Arquitectura estable y escalable, a partir de tecnologías líderes (Node.js, Python, PostgreSQL, RabbitMQ, NestJS, FastAPI, React, TypeScript).
